Kitchen Manager

Boise Code Works Capstone Project

Kitchen Manager is my group capstone project for Boise Code Works. We used a mixture of many frameworks but the main ones we used were Angular, Bootstrap and Firebase as our server. Kitchen Manager makes planning meals and shopping for a family easier. In our app the user has the ability to create recipies, plan out all meals for weeks in advance and create a grocery list based on the schedule.

Guess Who

Guess Who is a spin off of the classic 1970's game

To create this app I mainly used Boostrap and Angular. The game allows the user to guess if the chosen player has certain attributes and if the chosen player has the attribute it will flip over every player that doesn't and visa vera. I also implemented a counter that will prevent unlimited guesses of attributes to make the game more exciting.

You can view the code at my github here.

Dwarf Match

Dwarf Match is a simply memory game that will allow the user to turn two cards over at a time and then turn over after a few seconds. The user must remember where the cards are located and try to find a pair. The game keeps track of the user's guesses. I utalized javascript, Angular and Bootstrap for this project.

You can view the code at my github here.
